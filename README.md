### Installation on Windows

1. Clone repo into a new `nvim` directory in `~/AppData/Local`
2. Install the following (via chococlatey)
`choco install neovim`
`choco install ripgrep`
`choco install mingw`
`choco install nerd-fonts-hack`
`choco install gcc`

### Setup packages
1. Run `:PackerInstall`
## For TS dev 
2. `:MasonInstall`
* eslint-lsp
* ts-standard
* prettierd
